# t-rex-runner

The T-Rex Runner game, based on [t-rex-runner](https://github.com/wayou/t-rex-runner),
to run on Onion OS as a port using Pygame (Python 2.7).

![](./screenshot.png)

![](./screenshot_night_mode.png)

## Installation on Onion OS

Enable Ports in: `Apps` › `Package Manager` › `Verified`.

Copy the `Roms` folder in repository to the root of your SD card with Onion OS installed,
merging the folders.
